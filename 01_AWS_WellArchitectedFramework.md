## Pillars

- **Operational Excellence**  
  Operational excellence focuses on the ability to support and run workloads effectively, gain insights into operations, and continuously improve processes and procedures to deliver business value. For CloudOps engineers, operational excellence involves automating repetitive tasks, monitoring system health, and continuously improving operational procedures.
- **Security**  
  Security emphasizes taking advantage of cloud technologies to protect data, systems, and assets. CloudOps engineers make sure cloud resources are secure. They achieve this security by implementing appropriate access controls, encrypting data both at rest and in transit, and following security best practices to protect against potential threats.
- **Reliablity**  
  Reliability covers the ability of a workload to perform its intended function correctly and consistently as expected. CloudOps engineers implement fault tolerance and have contingency plans in place to make sure workloads remain reliable and accessible.
- **Performance efficiency**  
  Performance efficiency focuses on using computing resources efficiently to meet system requirements and maintain efficiency as demand changes. CloudOps engineers monitor resource utilization, implement auto scaling mechanisms, and optimize workloads to make sure architectures are running efficiently.
- **Cost Optomalisation**  
  Cost optimization emphasizes the ability to run systems to deliver business value at the lowest possible cost. CloudOps engineers analyze usage patterns to identify opportunities for cost savings. This analysis facilitates using strategies such as rightsizing resources, using reserved instances, and turning off unused resources.
- **Sustainability**  
  Sustainability emphasizes the ability to continually improve by reducing energy consumption and increasing efficiency across all components of a workload. CloudOps engineers contribute by implementing energy efficient practices and optimizing resource utilization.
